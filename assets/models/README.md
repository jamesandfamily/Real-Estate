# 3D Models

Place your 3D model files (GLB, GLTF) in this directory.

## Supported Formats

- `.glb` - Binary glTF (recommended for web)
- `.gltf` - glTF JSON format
- `.hdr` - High Dynamic Range environment maps
- `.exr` - OpenEXR environment maps

## Example Models

You can download free 3D models from:

- [Sketchfab](https://sketchfab.com/)
- [Poly Haven](https://polyhaven.com/models)
- [Google Poly](https://poly.google.com/) (archived)

## Usage

Add your models to `js/data/models.js` and load them in `js/modules/three-scene.js`.
